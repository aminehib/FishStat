FishStat (version simple)

Ce projet lit des fichiers CSV et affiche les poissons dans une interface JavaFX.

Format CSV attendu :
species,length,weight,size,infestationRate,content

Chargement automatique des datasets :
- processed_data_anisakis.csv
- mackerel.97442.csv
- merlu2018_75164.csv

Lancer l'application (Windows PowerShell) :
.\run.ps1

Lancer l'application (WSL / Git Bash) :
bash run.sh
